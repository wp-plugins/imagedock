== Installation ==

Installing "ImageDock" can be done either by searching for "ImageDock" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
2. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to ImageDock Settings and define your gallery
5. Add the gallery via a shortcode (example: [imagedock id="2" width="200px" height="200px"])
